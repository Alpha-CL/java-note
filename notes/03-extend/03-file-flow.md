## file stream

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/** file stream **/

    // base
    FileInputStream, FileOutputStream                       // bit
    FileReader, FileWriter                                  // character

    // extend
    BufferedInputStream, BufferedOutputStream               // bit
    BufferedReader, BufferedWrite                           // character


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


/** file array stream **/

    // byte[]
    ByteArrayInputStream, ByteArrayOutputStream
    
    // char[]
    CharArrayInputStream, CharArrayOutputStream


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//
```

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


1. 为什么需要存储到硬盘的文件中

    永久性保存信息, 数据持久化等


2. 常规流的优缺点 ?

    可以根据 (字节，字符, 行 等), 读/写 文件         // 不安全
    
    只能记录 字符串, 无法记录方法


3. 为何产生对象流 ?
    
    解决 java对象 的 读/写, 将 对象 记录在文件中永久性保存
    ( 将 对象 随机拆分为字节碎片 后 读/写 )


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


对象 序列化/反序列化 ?


- 对象序列化

    将一个完整的对象, 拆分为字节碎片, 记录在文件中


- 对象返序列化

    将文件中记录的对象 随机合并 后 组成一个完整的对象


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


对象序列化需满足条件


实现 Serializable 接口

    eg: public class Demo implements Serializable {}

    * Serializable: 示意性接口, 无成员, 仅起标记作用 


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```

#### file_flow vs buffer_file_flow

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


// 性能比较

    缓冲流 > 常规流


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//



            // file stream( 文件流 ): 将硬盘中的文件, 通过文件流读取到内存中
                
                FileInputStream, FileOutputStream               // bit
                FileReader, FileWriter                          // character
            

            + --------------------- +                                       + --------------------- +
            |                       |                                       |                       |
            |                       |                                       |    file1, file2 ...   |
```
