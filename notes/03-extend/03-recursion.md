``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////



    +                                   +                                   +                               +
    |                                   |                                   |                               |
    |   + ------------------------- +   |                                   |                               ｜
    |   |    x. invoking            |   |                                   |                               ｜
    |   |    x. execute             |   |                                   |                               ｜
    |   + ------------------------- +   |                                   |                               |
    |                                   |                                   |                               |
    |   + ------------------------- +   |                                   |                               ｜
    |   |    3. invoking            |   |                                   |                               ｜
    |   |    1. execute             |   |                                   |                               ｜
    |   + ------------------------- +   |                                   |                               ｜
    |                                   |                                   |                               |
    |   + ------------------------- +   |                                   |                               ｜
    |   |    2. invoking            |   |                                   |                               ｜
    |   |    2. execute             |   |                                   |                               ｜
    |   + ------------------------- +   |                                   |                               ｜
    |                                   |                                   |                               |
    |   + ------------------------- +   |                                   |                               |
    |   |    1. invoking            |   |                                   |                               |
    |   |    3. execute             |   |                                   |                               |
    |   + ------------------------- +   |                                   |                               |
    |                                   |                                   |                               |
    + --------------------------------- + --------------------------------- + ----------------------------- +


    // 递归
    
        invoking: 当找到出口后, 会根据进栈的顺序倒序执行
        
        execute: 先进栈的后执行，内存溢出( 因为递归需要找出口, 在出口没有找到之前会一直往栈中添加新的方法 )


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
