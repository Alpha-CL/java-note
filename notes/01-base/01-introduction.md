#### naming rule

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


// 规则


1) 字母: 区分大小写

2) 数字       // 0 ~ 9 都可以

    - 不能用于开头

3) 符号       // 尽量避免: 部分符号有特殊意义

    - $: 内部类
    
    - _: 私有的

4) 中文       // 不推荐: 字节处理需拆解, 可能引发 bug


//-------------------------------------------------------------------------------------------------------------------//


// 规约


1) 首字母大写

2) 大驼峰命名

3) 有意义的命名


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
