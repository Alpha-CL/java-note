## type conversion

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


// 数据类型转化           // 赋值时，若前面的字节位仅用于占位，则 赋值运算 会将部分占位字节位去掉

    - 基本数据类型: 大空间数据存储类型 可以直接存储 小空间数据存储类型
                  小空间数据存储类型 需要强制类型转换存储 大空间数据存储类型
                  ( 若转换过程中数值超过边界，可能会损失数据精度 )
    
        - 自动: 小字节类型 转换 大字节类型
        
            byte a = 1; int b = a;
            float a = 3.4F; double b = a;
        
        
        - 强制: 大字节类型 转换 小字节类型
        
            int a = 1; byte b = (byte)a;
            double a = 3.4; float b = (float)a;
        
        
        - 若 "小字节位存储的字节" 大于要转换 "大字节类型的字节数量"
        
            1) 使用强制转换
            
            2) 可以编译，但会丢失部分精度


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


    - 引用数据类型
    
        - 自动
        
        - 强制
```
