#### spring boot

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * spring boot
 * 
 * 
 * 用 约定 减少 配置
 */


- 创建 独立, 标准 的 spring 应用

- 直接内嵌 Tomcat/Jetty/Undertow 容器( 无需部署 war 包  )

- 提供 starter 依赖, 简化构造配置

- 尽可能自动化 spring 配置 和 第三方 依赖及管理

- 提供准生产环境的 特性( eg: 指标, 健康检查和外化配置 ) 


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


// 优势

    - 使编码更简单: 遵循 '约定 优于 配置' 原则
    
    - 使测试更简单: 内置7种测试框架( Junit/SpringBootTest/Assert/Hamcrest/Mockito/JSONassert/JsonPath )
    
    - 使部署更简单: 内嵌容器, 集成jenkins, 支持容器化
    
    - 使监控更简单: Actuator 监控应用健康, SpringBoot Admin 集群监控工具


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
