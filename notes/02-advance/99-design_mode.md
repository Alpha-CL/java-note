#### node mode

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * 节点模式: 单向链 | 双向链 | 环向链
 * 
 * 
 * 每一个几点都会包含
 * 
 *      - 上一个节点的地址
 *      - 下一个节点的地址
 *      - 当前节点的内容
 */

      + ----------------- +        + ---------------- +       + ----------------- +
      |                   |        |                  |       |                   |
      |     firstNode     | -----> |   ... Node ...   | ----> |     lastNode      |
      |                   |        |                  |       |                   |
      + ----------------- +        + ---------------  +       + ----------------- +
    
          prev: null,                 prev: firstNode             prev: Node,
          content: {}                 content: {},                content: {},
          next: Node                  next: lastNode              next: null


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
