# class-enum

#### introduction

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * enum class           // JDK 1.5 +
 * 
 * 
 * 默认直接继承 enumClass -> Enum -> Object           // 不可使用 extends 继承其他类
 */


//-------------------------------------------------------------------------------------------------------------------//


// 常用属性

    - name(): stringName;               // 枚举对象的名字
    
    - ordinal(): indexNumber;           // 当前枚举项的创建索引


// 常用方法

    - final valueOf(): curEnumItemValue;        // 当前枚举项的值
    
    - final values(): allEnumItemsValue;        // 所有枚举项的值
    
    - final compareTo(): int;                   // 比较两个枚举对象, 返回优先创建的枚举对象的索引

    - toString();                               // 将当前枚举项转换为字符串( 此方法可重写 )


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
